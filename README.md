# Canny Edge Detection (MATLAB)

This project implements the Canny Edge Detection algorithm using MATLAB.

The goal of this project is to understand how edge detection works in image processing and how the Canny algorithm identifies object boundaries.

Algorithm Steps

1. Gaussian smoothing for noise reduction
2. Gradient computation using Sobel operators
3. Non-maximum suppression to thin edges
4. Double threshold detection
5. Edge tracking by hysteresis

Tools

MATLAB
Image Processing Toolbox

Example Result

The algorithm successfully detects object boundaries while suppressing noise, producing clear edge maps.

Applications

Computer vision
Object detection
Medical image analysis
